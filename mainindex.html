Đang phát triển

Hướng dẫn sử dụng Dashboard

  1. Truy cập nhanh: Nhấn vào card dự án để mở trang tương ứng trong tab mới.
  2. Thêm dự án mới: Tạo file HTML mới (ví dụ: lab2.html), sau đó thêm card mới vào grid trong file này.
  3. Cập nhật trạng thái: Thay đổi class status-active thành status-coming cho dự án đang phát triển.
  4. Đếm số lượng: Hệ thống tự động đếm số dự án đã hoàn thành và sắp ra mắt.
  5. Responsive: Dashboard tự động điều chỉnh layout trên mọi thiết bị.

Mẹo: Bạn có thể sao chép cấu trúc card và chỉnh sửa nội dung để thêm dự án mới một cách nhanh chóng.
