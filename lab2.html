Khám phá sức mạnh của CSS Keyframes

Thực hành 5 bài tập animation nâng cao với @keyframes và infinite loop

1 Text chạy ngang (Marquee)

Chữ chạy liên tục từ phải sang trái với animation infinite

✨ CSS ANIMATION - KEYFRAMES - INFINITE LOOP - HỌC VIỆN CNTT BACHKHOA ✨

@keyframes moveText { 0% { transform: translateX(100%); } 100% { transform: translateX(-100%); } }

2 Hộp quay vòng (Rotate loop)

Khối vuông xoay 360° liên tục với infinite animation

Khối vuông đang xoay vô hạn:

Animation: rotate 3s linear infinite

@keyframes rotateBox { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } }

3 Ảnh xuất hiện dần (Fade-in sequence)

3 ảnh xuất hiện tuần tự với animation-delay khác nhau

Ảnh sẽ xuất hiện lần lượt với độ trễ 0.5s:

/* Animation delay cho từng ảnh */ .fade-image:nth-child(1) { animation-delay: 0s; } .fade-image:nth-child(2) { animation-delay: 0.5s; } .fade-image:nth-child(3) { animation-delay: 1s; }

4 Nút nhấp nháy (Call to Action)

Nút thay đổi màu sắc liên tục để thu hút sự chú ý

Nhấp nháy vô hạn để thu hút người dùng:

Nút thay đổi màu từ đỏ → cam → đỏ

@keyframes blink { 0% { background: #FF6584; } 50% { background: #FDCB6E; transform: scale(1.05); } 100% { background: #ff4500; } }

5 Slider ảnh tự động bằng CSS

Slider chuyển ảnh tự động sau mỗi 5s không cần JavaScript

Slider tự động chuyển ảnh mỗi 5 giây:

Mỗi ảnh hiển thị 5 giây, tổng chu kỳ 15 giây

@keyframes slide { 0%, 25% { transform: translateX(0); } 33.33%, 58.33% { transform: translateX(-33.33%); } 66.66%, 91.66% { transform: translateX(-66.66%); } 100% { transform: translateX(0); } }
